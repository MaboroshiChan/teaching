# if else 语句

# 定义
在编程中，我们通常需要去对已有的数据进行判断，从而设计接下来的程序要执行的操作。
这就是`if`和`else`语句的作用。
# 语法
```java
if(条件语句){
    代码块
}
```
# 例子1
如果你考试得了 100 分，星期天就带你去游乐场玩。
```java
if(你考试得了100分){
    星期天带你去游乐场玩;
}
```

# `while`语句
## 定义
在编程的过程中会遇到不少重复操作，`while` 语句就是用来打包这个操作的。

# 语法
```java
while(条件语句){
    代码块
}
```

```java
do {
    代码
}
while(条件语句)
```
